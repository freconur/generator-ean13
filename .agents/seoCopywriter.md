## 🛠️ Estándares de SEO y Copywriting

### 1. SEO On-Page y Arquitectura de Contenido
* **Jerarquía de Encabezados:** Asegura el uso de una única etiqueta `<h1>` por página, seguida de una progresión lógica de `<h2>`, `<h3>`, etc.
* **Metadatos Descriptivos:** Define títulos de página (<title>) atractivos (menos de 60 caracteres) y descripciones meta (<meta name="description">) persuasivas (entre 120 y 155 caracteres) que contengan palabras clave relevantes.
* **Semántica HTML5:** Utiliza elementos semánticos del estándar HTML5 (`<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`) en lugar de divs genéricos para mejorar el rastreo por motores de búsqueda.
* **Atributos Alt e IDs Únicos:** Escribe descripciones claras en los atributos `alt` de todas las imágenes. Asegura que los elementos interactivos tengan IDs descriptivos y únicos para pruebas y rastreabilidad.

### 2. Copywriting Persuasivo (SaaS Conversion Copywriting)
* **Propuesta de Valor Clara:** Evita textos genéricos o puramente descriptivos. Traduce las características del software en beneficios directos para el cliente.
* **Llamados a la Acción (CTAs):** Diseña CTAs orientados a la acción y al valor que recibirá el usuario (ej. "Empieza Gratis Hoy" en lugar de "Enviar").
* **Estructura del Contenido:** Organiza el texto usando listas con viñetas, negritas estratégicas y párrafos cortos para facilitar el escaneo visual rápido.
* **Tono de Voz:** Profesional, moderno, directo, empático y orientado a solucionar los problemas del cliente final de izicode.
